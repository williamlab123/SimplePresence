O SimplePresence é uma API que integra o seu app, ou seja, voce nao precisa baixar nada, nem usar nenhum site externo. A API do SimplePresence é adicionada diretamente no seu app e as funçoes sao executadas de dentro, diretamente entre o usuario e o servidor. Para entender melhor como funciona a API e como adiciona-la, clique AQUI. O Simple funciona de 3 formas diferentes, via QRCODE, WIFI e GEOLOCALIZAÇAO. Via QRCODE, por exemplo, o professor gera um QRCODE, transmite em um telao ou algo do tipo, entao os alunos, com a camera do celular, apontam e recebem a presença. Via WIFI, o professor inicia a chamada e todos os alunos conectados na mesma internet(LAN) conseguem registrar presença. Via GEOLOCALIZAÇAO, o professor abre a chamada e todos os alunos que estiverem num raio de x metros conseguem registrar presença. Em todos os passos, desde o professor iniciar a chamada até o aluno registrar, levam menos de 15 segundos. Prático, nao?
